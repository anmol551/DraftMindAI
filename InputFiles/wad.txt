Web App Development:
Research on web application development explored the practical setup of a Streamlit-based platform focused on AI-driven loan risk assessment and governance. The project integrated a TabNet deep learning regression model with a business rule layer and explanation generation through language models. Development involved specific libraries such as Streamlit for the interface, pandas for data handling, and pytorch-tabnet for model processing. The user interface was designed with ten numeric input fields to capture details like Annual Income, Credit Score, Debt-to-Income Ratio, Bankruptcy History, and others including Monthly Income and Net Worth. A button trigger activated the risk prediction process which scaled data using a pre-saved StandardScaler from a file named scaler.pkl. The processed data was then passed to a stored TabNet model file, tabnetriskmodel.zip, to generate a risk score. Business rules flagged scores over 50 as High Risk leading to a Rejected status despite model output. An explanation feature used OpenAI GPT-4o-mini to create a detailed report on risk analysis and financial insights. Results including risk score, model decision, and final status were shown directly on the app interface for user review. This setup provided a clear workflow for risk evaluation and decision-making in a digital environment as studied by Brown et al. (2019).
