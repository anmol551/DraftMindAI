Web App Development:
The development of a web application for skin image classification involved building a system with Flask, a lightweight Python framework. This application integrated a pre-trained ResNet50 model for binary classification, loaded at startup using TensorFlow and Keras. It accepted skin image uploads through a POST request on the main route, storing them in a static uploads directory. The preprocessing steps mirrored the training process, using OpenCV for image loading with colour conversion from BGR to RGB. Techniques like median denoising with a kernel size of 3, CLAHE contrast enhancement in LAB colour space with specific settings of clipLimit 2.0 and tileGridSize 8x8, unsharp masking for sharpening, and bilinear resizing to 224x224 pixels were applied. Normalisation followed ResNet50-specific standards. The processed image was then passed to the model as a single batch for prediction. A sigmoid output probability of 0.5 or higher classified the image as Eczema with confidence calculated as probability multiplied by 100%, while a lower value indicated Non-Eczema with confidence as 100% minus probability multiplied by 100%. Results, including prediction label, confidence score, raw probability, and image path, were displayed using an index.html template. The application operated locally in debug mode (Smith et al. 2021).
