The codebase implements an end-to-end NLP pipeline for automated resume classification and skill extraction, progressing from raw text loading to a deployable Flask web application. The project uses the 2,484-resume LiveCareer dataset from Kaggle and compares three models — Multinomial Naive Bayes, Bidirectional LSTM, and DistilBERT — across multiple preprocessing and class imbalance correction strategies.

The preprocessing pipeline includes contraction expansion, regex-based text cleaning, custom stopword removal preserving domain keywords, tokenisation, and lemmatisation. TF-IDF with unigrams and bigrams (5,000 features) provides structured feature representations for classical models, while DistilBERT operates directly on tokenised raw text. Named Entity Recognition using spaCy's PhraseMatcher extracts skills, education qualifications, and professional roles from each resume, generating auxiliary numeric features. The three models are trained on an 80/20 stratified split and evaluated using accuracy, macro precision, recall, F1-score, and a custom geometric mean metric to monitor class imbalance.

Class imbalance is systematically addressed through three strategies: sample-weighted training, SMOTE oversampling, and per-class decision threshold tuning. Weighted DistilBERT achieves the best overall performance at 83.50% accuracy and G-Mean of 0.7754, a dramatic improvement over the 0.2990 baseline. The Flask web application loads the saved DistilBERT model, processes uploaded PDF resumes through the full pipeline using PyMuPDF for text extraction, and returns a structured JSON response including predicted category, confidence score, per-class probabilities, and extracted entities — delivering a complete HR automation prototype.
