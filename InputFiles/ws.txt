The Flask web application serves as the deployment interface for the trained DistilBERT resume classification model. Built using Flask (Python), the app loads the saved DistilBERT model, tokenizer, label encoder (24 classes), and best per-class thresholds at startup via a custom load_model() function, operating on either CPU or CUDA depending on availability. The app accepts PDF resume uploads through the /predict POST endpoint (16 MB file size limit) and processes them through five sequential stages: PDF text extraction using PyMuPDF (fitz), full NLP preprocessing (contraction expansion, text cleaning, stopword removal, lemmatisation), DistilBERT tokenisation and inference using softmax probabilities and argmax classification, and spaCy PhraseMatcher-based named entity recognition for skills, education, roles, and organisations. The response returns a structured JSON payload containing the predicted job category, confidence score, all 24 class probabilities sorted descending, extracted entity lists, and a 500-character text preview. Additional endpoints include / (renders index.html template) and /health (returns model status and class list). Uploaded files are deleted immediately after processing for privacy.
