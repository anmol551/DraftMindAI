1. ABSTRACT
----------------------------------------
The accurate classification of resumes into job categories poses a significant challenge in recruitment due to varying formats and content. Machine learning algorithms provide an effective approach to automate this process improving efficiency. The study uses the Resume Dataset from Kaggle sourced from livecareer.com containing 2484 resumes across 24 categories such as IT and Business Development. The methodology includes preprocessing steps like contraction expansion lowercase conversion URL/email/phone removal special character removal tokenisation stopword removal and lemmatisation. Feature engineering incorporates TF-IDF vectorisation with n-gram range of 1-2 and maximum features of 5000 plus named entity recognition using SpaCy PhraseMatcher for skills education and roles. The models trained and evaluated are Multinomial Naive Bayes Bi-LSTM and DistilBERT. The best model Threshold-Tuned DistilBERT achieved an accuracy of 83.50 per cent. This surpassed the baseline MNB model accuracy. A web application using the Flask framework was developed to deploy the model enabling users to upload PDF resumes process them through text extraction and prediction then return the predicted category confidence scores and extracted entities. The application ensures privacy by deleting uploaded files post-processing.

Keywords: Resume Classification, Machine Learning, Distilbert, Bidirectional Lstm, Multinomial Naive Bayes, Text Preprocessing, Tfidf Vectorization, Spacy Phrasematcher, Kaggle Dataset, Flask Framework, Job Categories, Accuracy Score
