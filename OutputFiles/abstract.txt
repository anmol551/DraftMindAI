1. ABSTRACT
----------------------------------------
The financial sector encounters difficulties in loan approval processes due to high denial rates stemming from inaccurate risk assessment and class imbalance in data. Machine learning provides tools to enhance prediction accuracy by identifying key patterns in applicant financial profiles. This research employs a synthetic financial risk dataset from Kaggle with 20000 records and 37 columns including features like AnnualIncome CreditScore and LoanApproved. The methodology incorporates preprocessing steps such as dropping non-informative columns encoding categorical variables with OneHotEncoder and LabelEncoder engineering new features like IncomeStabilityRatio and DebtStressIndex through calculations based on existing data and selecting 10 features via Recursive Feature Elimination. Subsequent scaling of numerical features used StandardScaler to standardise the data before splitting it into training and testing sets. Models trained included ElasticNet a convolutional neural network and TabNet for both regression and classification tasks. TabNet emerged as the best model with an accuracy score of 89.1%. This performance surpassed the benchmark outlined in the dataset's original analysis. A web application built with Streamlit framework enables real-time loan risk evaluation by inputting user data processing it through the TabNet model applying business rules for decision overrides and generating detailed reports using language models for interpretation. The system processes inputs like AnnualIncome and CreditScore to output risk scores final decisions and explanatory reports enhancing practical application in financial governance. This approach addresses loan approval challenges effectively through data-driven insights. 

Keywords: Machine Learning, Loan Approval, TabNet, Convolutional Neural Network, ElasticNet, Feature Engineering, StandardScaler, Recursive Feature Elimination, Kaggle Dataset, Risk Prediction, Streamlit Framework, Classification Models
