Abstract
The growing challenge of identifying skin conditions like eczema through visual analysis has drawn attention in medical research due to the complexity and imbalance in available data. Machine learning offers a promising approach to automate and improve diagnostic accuracy for such conditions by learning patterns from images. This study uses the Skin Diseases Image Dataset sourced from Kaggle, which includes varied classes with a significant imbalance between eczema and non-eczema images. The methodology focuses on preprocessing steps such as converting image colour spaces, applying denoising techniques, enhancing contrast, sharpening details, resizing to a standard dimension, and normalising data for model compatibility. Models trained and assessed include a custom convolutional neural network, EfficientNetB0, and ResNet50, each designed to handle binary classification tasks. Among these, ResNet50 emerged as the top performer with an accuracy of 83.02%. This result surpassed the benchmark set by previous work in skin disease classification as reported by Ahmed et al. (2019). To extend practical application, a web-based tool was developed using Flask, a Python framework, allowing users to upload skin images for instant classification as eczema or non-eczema with confidence scores. This platform ensures seamless integration of the trained model for real-time predictions, bridging the gap between research and accessible healthcare solutions for early detection and management of skin conditions.

Keywords: Machine Learning, Skin Disease, Eczema Detection, Binary Classification, Image Preprocessing, Convolutional Neural Network, EfficientNetB0, ResNet50, Web Application, Flask Framework, Dataset Imbalance, Diagnostic Accuracy
